# 👕 Fashion-MNIST CNN Classification Project
> **2026학년도 1학기 인공지능개론 기말 팀 프로젝트**
> 본 프로젝트는 PyTorch 프레임워크와 밑바닥부터 구현하는 Scratch 방식을 통해 Fashion-MNIST 데이터셋을 정교하게 분류하는 CNN 모델을 구축하는 것을 목표로 합니다.

---

## 🎯 1. 프로젝트 목표 및 개요
* **핵심 미션:** Fashion-MNIST 데이터셋에 대해 최적의 분류 성능을 내는 CNN 아키텍처 설계.
* **학습 가치:** 
    * 프레임워크(PyTorch)의 효율성과 밑바닥(Scratch) 원리의 깊이 있는 이해 병행.
    * 하이퍼파라미터 튜닝 및 신경망 구조 변경을 통한 성능 최적화 경험.
    * 팀원 간의 협업, 일정 관리 및 실험 데이터의 합리적 입증 능력 함양.

---

## 🔬 2. 연구 내용 및 상세 구현 요구사항

### Track 1. PyTorch 기반 모델 구현 (5점)
* **목표:** 프레임워크를 활용한 고성능 CNN 아키텍처 설계.
* **평가 항목:**
    * 훈련 및 테스트 학습 진행 사항 그래프 제출 (2점).
    * **성능 기준:** 훈련 정확도 **96% 이상**, 테스트 정확도 **93% 이상** 달성 시 (3점).

### Track 2. Scratch 기반 모델 구현 (10점)
* **목표:** 『밑바닥부터 시작하는 딥러닝』 소스 코드를 응용하여 외부 API 없이 모델 구축.
* **제한 조건:**
    * 기존 프레임워크 API(PyTorch, TensorFlow 등) 사용 금지. 오직 교재 소스 응용.
    * 교재 4~8장의 이론 및 기법(Batch Norm, Dropout, He 초기값 등) 적극 활용 권장.
* **평가 항목:**
    * 학습 진행 사항 그래프 (교재 p.201 그림 6-9 참조) 제출 (2점).
    * **성능 기준:** 훈련 정확도 **95% 이상**, 테스트 정확도 **92% 이상** 달성 시 (8점).

---

## 📊 3. 데이터셋 정보
* **데이터셋:** Fashion-MNIST ([GitHub Source](https://github.com/zalandoresearch/fashion-mnist/tree/master/data/fashion))
* **특징:** 28×28 흑백 이미지, 10개 카테고리.
* **데이터 구성:** 학습용 60,000개 / 테스트용 10,000개.
* **주의사항:** 학습 시에는 오직 Training Set만 사용하며, Test Set은 최종 성능 평가 시에만 활용할 것.
